# Tweet: x402 Payments

Making AI agent infrastructure sustainable.

Bitcoin Yield Copilot integrates x402 payments for transparent, onchain compensation:
- Data feeds (prices, APYs) paid in sBTC/STX
- Each API call has a verifiable cost
- Users see exactly what they are paying for

The agent pays for what it uses - not a black box subscription.

This is how autonomous agents should work: every operation has a clear cost, every payment is verifiable onchain.
